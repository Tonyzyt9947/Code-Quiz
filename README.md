# Code Quiz

This project aims to create a timed coding quiz of multiple choice questions. The quiz ends if either all questions are answered or runs out of time. Users can choose to locally save their highscores and can view them on a highscore page.
​
## Getting Started
​
To open the deployed site, click [here](https://tonyzyt9947.github.io/PasswordGenerator/).

To view the html code, open html files.

To view the css code, open assets/CSS.

To view the javascript code, open assets/JS.
